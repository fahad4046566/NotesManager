# Smart Budget Manager

A Notes management app to manage expenses with category-wise limits.

## Features
- Add, Edit, Delete Notes
- Set category-wise Notes
- Filter using search notes by category
- LocalStorage persistence
- Responsive design (mobile + desktop)

## Tech Stack
- React + Vite
- Tailwind CSS + DaisyUI
- React Icons
